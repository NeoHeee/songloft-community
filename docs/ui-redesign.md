# Songloft UI Redesign

本分支用于在保留 Songloft 现有后端、播放、认证和插件逻辑的前提下，重构客户端视觉与交互。

## 第一阶段（已完成）

- 重建亮色与暗色主题：紫蓝强调色、沉浸式背景、统一圆角与卡片层级。
- 重做移动端迷你播放器：悬浮卡片、加大封面、下一首快捷操作、底部进度条。
- 重做桌面端播放器：悬浮播放坞、强化歌曲信息区和控制区、保留原工具栏能力。

## 第二阶段（已完成）

- 新增桌面/平板新版应用外壳。
- 桌面端采用品牌侧边栏 + 内容区 + 播放队列 + 底部播放坞布局。
- 平板端使用紧凑图标导航，保持更多内容空间。
- 重构首页欢迎区、音乐库状态、快捷入口、歌单网格和封面占位样式。
- 保留插件页面在 Web 端的 Offstage 保活机制。

## 第三阶段（已完成）

- 重构歌曲库类型筛选栏和歌曲数量显示。
- 重构移动端歌曲卡片、桌面歌曲行、类型标签与当前播放状态。
- 统一导航图标为新版圆角风格。
- 重构歌单网格卡片与列表视图，保留播放、多选、编辑、隐藏和删除能力。
- 重构设置页移动端入口、桌面分类侧栏和设置分组卡片。
- 重构插件快捷入口网格。

## 第四阶段（已完成）

- 重构歌曲库顶部搜索和操作面板。
- 修复搜索框清除按钮不能即时刷新的问题。
- 重构桌面/平板全屏播放器，采用封面信息卡、沉浸歌词和玻璃控制台布局。
- 重构歌词加载、空状态、失败状态和当前歌词焦点样式。
- 重构歌单详情页：宽屏采用歌单信息侧栏与歌曲工作区，移动端采用沉浸封面卡片。
- 重构歌单内搜索、播放、添加、多选、批量删除和永久排序操作。
- 重构插件管理：状态统计、搜索筛选、响应式卡片、上传、更新、强制安装和清理残留。

## 第五阶段（已完成）

- Android 启动页从默认纯白背景改为随系统明暗模式切换的紫色渐变，并复用现有应用图标。
- Web/PWA 浏览器主题改为新版紫色体系，补充 `theme-color`、深浅色声明和 iOS Web App 元数据。
- PWA 取消强制竖屏，支持手机和平板横竖屏自适应。
- 新建服务端 `ui-redesign-integration` 分支和集成工作流。
- 使用固定前端 Commit 构建嵌入式 Flutter Web、Go 完整版二进制和 Docker 镜像。
- Docker 容器启动后验证首页、部署模式、主程序 JS 与 CanvasKit 均可访问。
- 输出可通过 `docker load` 导入的 NAS 测试镜像包。

## 第六阶段：自定义主题包（已完成）

- 将“浅色 / 深色 / 跟随系统”与主题包选择分离，同一主题包可同时提供浅色和深色配色。
- 内置 Songloft 经典、深海蓝、森林绿和暮色玫瑰四套主题。
- 支持导入声明式 `.songloft-theme` / JSON 主题包，并进行版本、ID、颜色、圆角和文件大小校验。
- 导入前显示主题名称、作者、版本、主题 ID、描述及浅色/深色预览。
- 检测到相同自定义主题 ID 时明确提示覆盖更新，确认后才写入并启用。
- 支持查看主题详情、复制标准 JSON、导出指定主题以及一键导出当前主题。
- 支持删除自定义主题，内置主题保持只读。
- 完善主题包制作、发布、命名、版本更新和安全规范。
- 增加主题包解析、默认值、导出回环和非法输入自动测试。

## 第七阶段：在线主题目录（已完成）

- 在“设置 → 外观设置 → 主题包”中增加“在线主题”入口。
- 新增适配手机、平板和桌面的主题目录界面。
- 支持按名称、作者、简介、主题 ID 和标签搜索，并支持标签筛选。
- 主题卡同时展示浅色与深色预览、作者、版本、简介、标签和精选状态。
- 自动识别未安装、已安装和可更新状态。
- 远程目录不可用时自动切换至安装包内置安全快照。
- 首批提供“极光紫”和“暖纸琥珀”两套社区主题，并随各平台安装包打包。
- 远程地址限定为受信任 HTTPS 域名和仓库路径，禁止重定向、查询参数、认证信息和任意第三方来源。
- 下载后校验 SHA-256、主题 ID、名称、版本和作者，校验通过后仍需用户确认安装。
- 增加目录解析、信任域名、路径穿越、摘要篡改和身份不一致自动测试。
- 当前安全模型是受信任来源白名单和文件完整性校验，尚未宣称具备公钥数字签名。

## 设计原则

- UI 与业务状态解耦，尽量复用现有 Riverpod Provider。
- 不改变现有 REST API 和插件协议。
- 桌面、平板、手机和 TV 保持响应式能力。
- 主题包只允许声明式白名单字段，不执行脚本、CSS、HTML 或远程代码。
- 在线资源必须通过来源、大小、摘要和身份校验，并由用户主动确认安装。
- 每个阶段独立提交，便于回滚和跟进上游更新。
